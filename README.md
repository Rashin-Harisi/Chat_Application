# 💬 Real-Time Chat Application (WIP)

🚧 **Work in Progress** 🚧

This project is a full-stack **real-time chat application** that I am currently building as part of my learning journey and portfolio development.
The goal is to design and implement a scalable, modern messaging system using industry-standard technologies.

---

## 🎯 Project Goal

* Build a **production-ready chat system**
* Practice **real-time communication**
* Strengthen skills in **full-stack JavaScript development**
* Prepare for **junior/entry-level developer roles**

---

## 🧠 Tech Stack

### 🗄️ Backend

* Node.js + Express
* MongoDB (Atlas)
* Mongoose
* Socket.IO (real-time communication)
* JWT Authentication
* Swagger / OpenAPI (API documentation)

### 🎨 Frontend

* React
* TypeScript
* Zustand (state management)
* Socket.IO Client
* Axios
* Tailwind CSS
